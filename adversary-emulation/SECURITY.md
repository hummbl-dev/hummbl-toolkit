# Security Policy

## Supported Versions

This repository is actively maintained. Security updates are provided for the
latest release on the default branch.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately via email to
**security@hummbl.io**. Do not open a public issue.

We aim to respond within 48 hours and will coordinate a responsible disclosure
timeline with you.

## Scope

This policy applies to all code, documentation, and infrastructure within the
hummbl-dev organization.
