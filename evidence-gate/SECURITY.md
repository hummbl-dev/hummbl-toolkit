# Security Policy

## Reporting a Vulnerability

If you discover a security issue in Evidence Gate or its rule library, email `reuben@hummbl.io` with:

- affected file or script
- reproduction steps
- expected and observed behavior
- potential impact

Do not open public GitHub issues for suspected vulnerabilities.

## Scope

This policy covers the TOML rule library, fixture harness, validation tooling, and release artifacts in this repository.
