# License and Publication Posture

## Current Posture

This repository does not currently declare a root `LICENSE` file or an open-source license grant.

Until HUMMBL makes an explicit publication decision, treat this repository as:

- HUMMBL-owned material;
- not redistributable outside HUMMBL without written permission;
- not licensed for third-party reuse;
- not ready for package publication or external mirroring based on repository visibility alone.

## Why There Is No LICENSE File Yet

Adding a root `LICENSE` file creates an affirmative reuse signal. That signal should not be added until the operator chooses the intended publication posture for this repository.

## Decision Options

Before publication, choose one of:

- keep the repository unlicensed with this posture notice;
- add a proprietary/internal-use license notice;
- publish selected material under an open-source license;
- split public-safe material into a successor repository and keep this repository internal.

## Current Rule

Do not add a root `LICENSE` file, open-source license metadata, or package publication enablement without an explicit operator decision.
